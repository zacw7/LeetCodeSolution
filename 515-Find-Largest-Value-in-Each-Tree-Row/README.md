# 515. Find Largest Value in Each Tree Row

**Medium**

[Original Page](https://leetcode.com/problems/find-largest-value-in-each-tree-row/)

You need to find the largest value in each row of a binary tree.

##### Example:
```
Input: 

          1
         / \
        3   2
       / \   \  
      5   3   9 

Output: [1, 3, 9]
```
